# Dog vs Cat Classification using CNN

## Overview

This project implements a Convolutional Neural Network (CNN) using PyTorch to classify images of dogs and cats. The model learns visual patterns and features from images to accurately distinguish between the two animal categories.

## Features

* Binary Image Classification
* Custom Dataset Handling
* CNN-based Feature Extraction
* Training and Validation Pipeline
* Accuracy and Performance Evaluation
* PyTorch Deep Learning Implementation

## Technologies Used

* Python
* PyTorch
* Torchvision
* NumPy
* Matplotlib
* Scikit-learn

## Dataset

The dataset contains images of:

* Dogs
* Cats

The images are divided into training and validation sets for model evaluation.

## Model Architecture

* Convolutional Layers
* ReLU Activation
* Max Pooling Layers
* Fully Connected Layers
* Softmax Output Layer

## Results

The CNN model learns distinguishing features such as fur patterns, facial structure, and body shape to classify dog and cat images with high accuracy.

## Applications

* Pet Recognition Systems
* Computer Vision
* Image Classification
* Deep Learning Research
